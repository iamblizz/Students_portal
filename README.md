A simple Rest API built with Nod3.js and Express to manage student records using an in-memory database
(JavaScript array). Buit and tested.

## Features:
**POST** `/students`  - Add a new student
**GET** `/students`   - Get all students
**GET** `/students/:id`  - Get a student by ID
**PUT** `/students/:id`  - Update a student
**DELETE** `/students/:id` - Delete a student

## Tech Stack
Node.js
Express.js
